# Changelog

### Добавлено
- Первый релиз PlayerokCardinal
- Автоматические ответы на сообщения
- Автоматическая выдача товаров
- Автовосстановление лотов
- Управление через Telegram бота
- Поддержка Linux (Ubuntu, Debian) с systemd
- Поддержка Windows
- Установочный скрипт для Linux (install-poc.sh)
- Systemd service файл
- Первичная настройка (first_setup.py)
- PlayerokAPI для работы с платформой
- Система логирования
- Поддержка прокси
- Безопасное хранение паролей (bcrypt)
- Чистый код без комментариев

### Дизайн
- Архитектура основана на FunPayCardinal
- Упрощенная и оптимизированная структура
